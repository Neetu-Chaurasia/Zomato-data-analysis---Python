### Project Overview

This project analyzes Zomato restaurant data using Python to uncover insights about ratings, votes, prices, cuisines, and restaurant popularity.

### Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

### Key Analysis

* Relationship between votes and ratings
* Price comparison of high-vote vs low-vote items
* Cuisine-wise and restaurant-wise insights
* Popularity and pricing trends

### Insights

* Votes are not a strong predictor of ratings
* High-vote items are not always more expensive
* Other factors influence customer ratings and popularity

### Purpose

To practice data cleaning, exploratory data analysis (EDA), and visualization using real-world food delivery data.
